# App Wall 应用墙页面 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 app-harmony-wall 仓库构建一个静态应用墙页面，列出所有已上架鸿蒙 APP，通过 GitHub Pages 托管，供各 APP 内"更多应用"入口跳转使用。

**Architecture:** 单文件 `index.html` 通过 `fetch` 读取同目录 `apps.json` 动态渲染卡片列表；APP 数据与页面模板分离，新增/下架 APP 只需编辑 `apps.json`；图标文件统一管理在 `icons/` 目录下，图标源文件来自各 APP 工程的 AppScope 目录。

**Tech Stack:** 纯静态 HTML + CSS + Vanilla JS（无框架、无构建工具）；GitHub Pages 托管。

---

```
app-harmony-wall/
├── index.html            # 创建：页面模板 + CSS + JS 渲染逻辑
```

```html
      font-size: 12px;

    }

    .card-desc {
      font-size: 14px;
      color: #666666;
      line-height: 1.55;
    }

    .card-footer {
      margin-top: 14px;
      text-align: right;
    }

    .btn {
      display: inline-block;
      background: #1a1a1a;
      color: #ffffff;
      font-size: 14px;
      font-weight: 500;
      padding: 9px 20px;
      border-radius: 20px;
      text-decoration: none;

    }

      opacity: 0.75;

  <div class="page">
    <h1>我的鸿蒙应用</h1>
    <div id="app-list"></div>
  </div>

  <script>
    fetch('apps.json')
      .then(function(r) { return r.json(); })
      .then(function(apps) {
        var list = document.getElementById('app-list');
        list.innerHTML = apps.map(function(app) {
          var tagsHtml = app.tags.map(function(t) {
            return '<span class="tag">' + t + '</span>';
          }).join('');

          var url = 'https://appgallery.huawei.com/app/detail?id=' + app.bundleName;

          return '<div class="card">'
            + '<div class="card-top">'
            + '<img class="card-icon" src="' + app.icon + '" alt="' + app.name + '">'
            + '<div class="card-info">'
            + '<div class="card-name">' + app.name + '</div>'
            + '<div class="tags">' + tagsHtml + '</div>'
            + '<div class="card-desc">' + app.description + '</div>'
            + '</div>'
            + '</div>'
            + '<div class="card-footer">'
            + '<a class="btn" href="' + url + '" target="_blank" rel="noopener">在应用市场查看</a>'
            + '</div>'
            + '</div>';
        }).join('');
      });
  </script>
</body>
</html>
```

打开 `http://localhost:8080`，逐项检查：

1. 页面标题显示"我的鸿蒙应用"
2. 共 6 张卡片，顺序：密语匣子、决定今天吃什么、聚沙攒钱、呼吸视界、像素征途、订阅斩

5. 将浏览器窗口缩窄至 375px（模拟手机），卡片正常单列显示，文字无溢出

> 若图标显示破图，检查 icons/ 目录下对应文件是否存在，以及 apps.json 中 icon 字段路径是否正确。

- [ ] **Step 3: 停止本地服务器**

在终端按 `Ctrl+C` 停止服务器。

## 后续维护：新增 APP 时的操作

新 APP 上架后，只需两步：

1. 复制图标到 `icons/`：
   ```bash
   cp ../新APP工程/AppScope/resources/base/media/foreground.png icons/新app名.png
   ```

2. 在 `apps.json` 末尾追加一条记录：
   ```json
   {
     "name": "APP中文名",
     "bundleName": "com.xun.xxx",
     "description": "一句话简介",
     "tags": ["标签1", "标签2"],
     "icon": "icons/新app名.png"
   }
   ```

3. 提交推送，页面自动更新。
